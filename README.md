# tmux-AI-Orchestration-skill_claude-code_codex

## 概要
ブランチで Claude Code 向け / Codex 向け を分けています。

## 使い方
- それぞれの skill があるパスに配置する
- ブランチを切り替えて使う

## メモ
- Codex を主にして影の権能ごっこすると挙動が微妙（できなくはない）
- 個人的には Claude Code を主にした影の権能ごっこを推奨
  - アンソロピックの開発でそもそも並列使用されているので、当然ながら作りがそうなってるのは Claude Code
  - もすかすると、Claude Code でskill作ってもらって、それおCodexに移植したからようすがおかしいのかもしれない。
  - codexでcodexによるcodexのためのtmuxAIオーケストレーションskillつくってみて具合がよかったらそちらでよいかと。
- すべて2026/01/16時点のinfo
